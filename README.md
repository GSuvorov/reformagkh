reformagkh
==========

Граббер для сайта Реформа ЖКХ (http://www.reformagkh.ru), сохраняет данные в формате CSV.
Данные выкачанные граббером для Москвы в 2014 г.

В новой версии граббера пока поддерживаются не все переменные.
